# Templates

Copy-ready and annotated templates for bootstrapping and improving agents.

## Contents

- [skeleton/](skeleton/) — Complete Phase 1 agent scaffold. **Read when:** running `bootstrap` or copying a new agent structure.
- [fragments/](fragments/index.md) — Annotated component templates (SYSTEM.md, skill, tool, permissions, index.md). **Read when:** generating or improving a specific file type.

## When to read

- **Bootstrap:** start with `skeleton/` — copy entire directory to target path.
- **Improve one component:** read `fragments/index.md` for the relevant fragment only.
